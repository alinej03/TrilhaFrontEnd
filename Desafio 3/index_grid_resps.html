Respostas

A)O GRID é como se fosse uma grade, com isso podem ser usadas para o design de layouts de grandes seções de uma página da web, assim como pequenos elementos de interface.

B) GRID container são elementos que ficam dentro do item pai:

  • Display:Define o elemento como um grid container;
  • Para colunas usamos o grid-template-columns;
  • Para linhas usamos o grid-template-rows.

C)Os Grid Itens são os filhos diretos do Grid Container. Um grid item pode ser explicito ou implícito. Explicito é quanto você define ele explicitamente no container e implícito é quanto ele é criado automaticamente para preencher o conteúdo no grid.

D)A unidade fr representa uma fração do espaço disponível no container do grid. A próxima definição de grid cria três espaços com tamanhos iguais que aumentam e diminuem de acordo com o espaço disponível

E)São usada para criar uma repetição de listagem de faixas quando a grade é grande

F)Podemos observar que quando definimos um grid definimos como trilhas do grid, não como linhas. O grid então nos devolve linhas numeradas para usarmos aos itens posicionados

G)Uma célula de grade é a menor unidade em uma grade. Conceitualmente é como se fosse uma célula de tabela,

H)Itens podem se espalhar por uma ou mais células em ambas as linhas ou colunas, e isto cria uma área de grade. Áreas de grade devem ser retangulares – não é possível criar uma área em L por exemplo. Uma área destacada se espalhada por duas trilhas de linhas e duas trilhas de colunas.
